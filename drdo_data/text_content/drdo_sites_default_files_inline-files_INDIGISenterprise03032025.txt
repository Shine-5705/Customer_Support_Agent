INDIGIS-Enterprise 

INDIGIS-Enterprise  is   a   Military   Enterprise   Geographical   Information   system
indigenously   developed   by   CAIR,   DRDO   which   can   provide   a   platform   for
visualization   and   analysis   of   digital   maps   and   overlays   to   carryout   operation
planning   for   tri-services.   It   offers   OGC   certified   service   APIs   and   Applications
developed using this technology will be DCyA certifiable.   Along with the service
APIs, it also offers a suite of desktop APIs using which custom applications can be
developed to meet specific requirements of armed forces. 
The offered technology consists of INDIGIS-Server, INDIGIS Web Client, INDIGIS
Desktop and INDIGIS Mobile App along with Software Development Kit (SDK). It
has judicious mix of GIS-Remote Sensing-Photogrammetric analysis with 2D and
3D visualization. It can support a geo-database with hundreds of TBs of data and
services can be offered to large number of clients over limited bandwidth network.
This can become Import substitute & it possesses Export Potential. Salient features
of this technology are
Architectural:
 Scalable   architecture,   scale   agnostic   performance,   deployable   on   any   military

network

 Bandwidth adaptive GIS services
 Available on Desktop, Web, Mobile
 Centralised map repository, Decentralised publishing & integrated visualisation 

Functional:
 Military Grid & Military Decision Aids
 Offline and Online Capability in Desktop, No data residue in web client cache
 Amphibious & Cross Domain Capability
 Capability to handle MIL and Open Source Data at world scale
 State-of-the art Capabilities in comparison with existing COTS and OSS
 Suite of customizable and reusable GIS library

Certifications: 

 OGC Certified Interoperable Military GIS
 CERT-in certified Military GIS
 DCyA certifiable
